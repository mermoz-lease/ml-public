# 📦 Mermoz Lease — Public Assets

Ce repository contient les ressources publiques de **Mermoz Lease**, librement accessibles pour une intégration dans des sites web, applications partenaires ou tout support de communication externe.

---

## 📁 Structure du repository

```
ml-public/
└── logos/                     # Logos publics en accès libre
```

---

## 🖼️ Logos

Le dossier [`logos/`](logos/) contient les versions des logos Mermoz Lease autorisées pour un usage public. Ces fichiers peuvent être utilisés directement via leur URL raw GitHub pour une intégration simplifiée.

### Exemple d'intégration (HTML)

```html
<img
  src="https://raw.githubusercontent.com/mermoz-lease/ml-public/main/logos/<nom_du_fichier>.png"
  alt="Mermoz Lease"
/>
```

### Exemple d'intégration (Markdown)

```markdown
![Mermoz Lease](https://raw.githubusercontent.com/mermoz-lease/ml-public/main/logos/<nom_du_fichier>.png)
```
